# GitHub Portfolio Kit

Colección de plantillas y herramientas para estandarizar y profesionalizar perfiles de desarrolladores en GitHub.

## 📦 Contenido
*   **Plantillas de README:** Para diferentes tipos de proyectos (Web, Data Science, Librerías).
*   **Workflows:** Configuraciones de GitHub Actions para CI/CD.
*   **Guías:** Buenas prácticas de documentación y contribución.

## 🚀 Uso
Copia los archivos de la carpeta `templates` a tu nuevo repositorio y adáptalos a tus necesidades.
