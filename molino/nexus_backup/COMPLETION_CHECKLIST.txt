🚀 CÓMO USAR AHORA

1. Abre http://localhost:9002
2. En Sidebar, haz clic en "Primavera Analysis" (⠞ BarChart4)
3. Sube un archivo Excel o usa datos de demostración
4. El parser automáticamente:
   ✓ Detecta columnas técnicas
   ✓ Mapea a PrimaveraTask
   ✓ Extrae WBS (áreas)
   ✓ Calcula ruta crítica
   ✓ Identifica frentes

5. Usa filtros para:
   • Ver solo ruta crítica (Slack = 0)
   • Filtrar por frentes de trabajo
   • Buscar tareas
   • Rangos de fechas

6. El grid virtualizado maneja:
   ✓ 10,000+ tareas sin lag
   ✓ Scroll suave a 60fps
   ✓ Memory eficiente
