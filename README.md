# Repositorio de Scripts Python

Este repositorio contiene una colección organizada de scripts de Python, proyectos y ejercicios, clasificados por temática.

## Estructura del Proyecto

*   **📂 cursos/**
    *   `grua_python/`: Material completo del curso de Python (Capítulos 01-11), cubriendo desde fundamentos hasta métodos numéricos y análisis de datos.

*   **📂 ingenieria/**
    *   `dinamica_mecanismos/`: Simulaciones de dinámica y mecanismos (Manivela, Cuatro Barras, etc.).
    *   `modelos_matematicos/`: Aplicaciones estocásticas y modelos matemáticos.
    *   `poleas_bombas/`: Cálculos de ingeniería para poleas y sistemas de bombeo.

*   **📂 proyectos_web/**
    *   `mantenimiento_app/`: Aplicación web (Flask) para gestión de mantenimiento.
    *   `ANTIGRAVITY/`: Aplicación web de aprendizaje de Chino (React/Vite).

*   **📂 playground/**
    *   `scripts_sueltos/`: Scripts varios y pruebas (LibreOffice, etc.).

## Requisitos

Las dependencias principales del proyecto se encuentran en `requirements.txt`. Para instalarlas:

```bash
pip install -r requirements.txt
```

## Notas

*   Los nombres de archivo en la carpeta `ingenieria` han sido normalizados a `snake_case` para mejorar la compatibilidad y legibilidad.
*   Se recomienda usar un entorno virtual (`.venv`) para ejecutar los proyectos.
